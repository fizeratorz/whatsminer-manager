# ⛏️ Whatsminer Manager

Веб-панель для мониторинга и управления майнерами Whatsminer через официальный API v3.0.x.

## ✨ Возможности
- 📊 Мониторинг хешрейта, температур, вентиляторов, потребления
- 🔗 Отображение пулов и воркеров
- ⚠️ Детектирование ошибок с расшифровкой кодов
- 🔄 Управление режимом работы (low/normal/high) и перезагрузка
- 👤 Авторизация по логину/паролю
- 📱 Адаптивный интерфейс для десктопа и мобильных
- 🗂️ Группировка майнеров по локациям/адресам

## 🚀 Быстрый старт с Docker

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/Fizeratorz/whatsminer-manager.git
cd whatsminer-manager
```
# 2. Соберите и запустите контейнер в фоне
```bash
docker compose up -d --build
```
